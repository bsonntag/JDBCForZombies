JDBC For Zombies - V1
=====================

Instruções
----------

- Criar as seguintes classes:
    - Zombie:
        - id
        - name
        - graveyard
    - Tweet:
        - id
        - zombie
        - text
        - publishDate
- Criar repositórios para zombies e tweets, extendendo a classe AbstractRepository.
- Implementar os métodos findAll, find, save e delete.
